MoonCake Responsive Admin Template

MoonCake is a responsive, full featured admin template build exclusively for sale at ThemeForest.

This template is build on Bootstrap framework using HTML5 and CSS3. Its main focus is simplicity while providing the most commonly used elements and widgets on web apps across devices and browsers.

Since v1.1, this template has two build in design styles integrated; Simple and Creative. Simple design targets users that desire simplicity, friendly and plain interfaces where only functionality is the main concern, while the creative design targets users that desire a sophisticated, graphical user interface with skeuomorphic elements.
